Added getc : unit -> int built-in to the OCaml interpreter. Handled a subtle Pascal-runtime lookahead quirk where the source's 0x04 EOT leaks into the first runtime read — getc discards a leading 0x04. Supporting: OCAML_STDIN env in run-ocaml.sh for feeding runtime input, demo-echo justfile target, eval_getc_echo test registered in reg-rs (passing), docs/stdin-and-getc.md explaining the wire protocol and the quirk.
